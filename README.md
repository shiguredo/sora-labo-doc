## 今後

### TURN-TCP 利用強制機能

指定した接続が TURN-TCP を利用するように強制可能にする機能です。

### TURN-TLS 利用強制機能

指定した接続が TURN-TLS を利用するように強制可能にする機能です。

### 帯域制限機能

指定した接続の帯域を制限する機能です。

### パケロス機能

指定した接続のパケロスの割合を指定する機能です。
